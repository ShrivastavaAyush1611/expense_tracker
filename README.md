## Getting Started

### Prerequisites

- Node.js (v18 or later)
- MongoDB Atlas account or local MongoDB
- npm or yarn

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/expense-tracker.git
   cd expense-tracker
2. Install dependencies
   ```bash
   npm install

     npm run dev
